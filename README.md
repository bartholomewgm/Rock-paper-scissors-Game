# Игра Камень-Ножницы-Бумага

## Описание

Консольная игра на Python, в которой пользователь играет против компьютера.
Реализован подсчет очков и сохранение результатов в файл.

## Как играть

1. Убедитесь, что у вас установлен Python 3.
2. Запустите игру командой:

   python game.py

3. Введите один из вариантов:
   - rock
   - paper
   - scissors
4. Для выхода из игры введите `exit`.

## Функциональность

- Игра против компьютера
- Проверка корректности ввода
- Подсчет побед игрока и компьютера
- Отображение текущего счета
- Сохранение результатов в файл `scores.json`

## Требования

- Python 3.x
